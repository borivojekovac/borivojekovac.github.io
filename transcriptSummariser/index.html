Transcript Summariser Bookmarklet

Drag and drop the link below to your bookmarks.

While on Microsoft Stream, click the bookmark, and then click the transcript you want summarised.

First time you use the script on a domain, it will prompt you for an OpenAI API key. From then on will store it in a cookie and reuse.

This Bookmarklet is specifically designed to pick out Microsoft Stream transcripts, but if that's not found when the text is clicked on, it will pick out innerText of the element clicked.
